# Contributing

## Development Commands

```bash
make install
make check-all
make docs
```

## Contribution Focus

Useful contributions include:

- improving recipe clarity
- adding or refining example projects
- tightening architecture guidance
- strengthening repository tooling and validation

## Expectations

- Keep all content in English.
- Keep recipe guidance practical and production-aware.
- Update documentation when public guidance changes.
- Keep tests and checks green before merge.
