# Azure Functions Python Cookbook

Azure Functions Python Cookbook is a practical collection of recipes for building real-world serverless applications with Azure Functions and Python.

This repository is designed to help developers:

- discover a proven implementation pattern
- understand the architecture behind it
- start from a clean baseline quickly

## What You Will Find

- curated recipes for common Azure Functions scenarios
- reusable example structures
- notes on pitfalls and production considerations
- scaffold-ready guidance for starting new projects

## Initial Focus

The initial recipe set covers:

- HTTP API Basic
- HTTP API with OpenAPI
- GitHub Webhook Receiver
- Queue Worker
- Timer Scheduled Job

## Repository Direction

The cookbook is content-first. It does not try to be a separate CLI product in the first release. The priority is to make recipes accurate, useful, and easy to extend.
