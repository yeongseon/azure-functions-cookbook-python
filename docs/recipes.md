# Recipes

The cookbook starts with a small set of focused recipes that map to common Azure Functions workloads.

## Planned MVP Recipes

- HTTP API Basic
- HTTP API with OpenAPI
- GitHub Webhook Receiver
- Queue Worker
- Timer Scheduled Job

## Recipe Contract

Every recipe should cover:

- overview
- when to use
- architecture
- project structure
- local run instructions
- production considerations
- scaffold guidance when available

The source recipe files live in the repository-level `recipes/` directory.
