# Security

## Reporting a Vulnerability

If you discover a security issue in this repository, please report it privately to the maintainer instead of opening a public issue.

Include:

- a short description of the issue
- the affected files or area
- reproduction details if relevant
- suggested mitigation if you have one

## Scope

This repository is primarily documentation and example oriented. Security concerns are most likely to relate to example code, generated snippets, dependency hygiene, or deployment guidance.
